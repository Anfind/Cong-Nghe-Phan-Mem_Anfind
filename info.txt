CẦN LÀM:
1. Website (ReactJS đơn giản)

Trang đăng nhập/đăng ký

2. Backend API (.NET Core hoặc Node.js)

Đăng nhập/đăng ký user

Quản lý bài thi (làm bài + chấm điểm tự động)
Lưu kết quả học tập

3. Database (MySQL)

Bảng user, khóa học, bài thi, kết quả
Thiết kế database đơn giản

4. Tài liệu

File README hướng dẫn chạy code
